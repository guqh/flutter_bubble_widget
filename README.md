# flutter_bubble_widget
flutter 库，纯 dart 气泡控件


在flutter工程中，这是一个强大的全方位的尖角的气泡组件，支持全部终端
Flutter小部件，用于创建不同类型的聊天气泡。您可以使用此小部件的不同属性并创建漂亮的聊天气泡，也可以用作弹窗背景。
